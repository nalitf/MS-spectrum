# MS-spectrum

MALDI-TOF-MS（およびGC-MS）の測定データからグラフを出力する  
分析センターのAB SCIEX, TOF/TOF(TM) 5800systemによって出力されるMALDI-TOF-MSスペクトルをPC上で表示したくて作成した  
pandasでtxt形式の測定データを読みこみmatplotlibでグラフ出力

### 今のところの改善すべき点
○読み込むときにヘッダ2行を固定値としてskipしているがこの部分はいつか自動でskipする行数を読み込めるようにしたい  
○現行ではソースコードを直接書き換えてファイルを指定しているけれど将来的にはGUIでファイル指定とかできるようにしてみたい  
○各測定値に対して1本ずつ棒グラフを書く形で描画しているため非常に動作が重い  
　これについては今一つ改善方法が分からない  
（折れ線形式で単純にplotすると速いけれどグラフとして適切なのかどうかが微妙、ただ高分解能なのでよほど狭い範囲を拡大しない限り両者の違いはあまり判別できない）
